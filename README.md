# 白酒品评数据的数学建模与可靠性分析思路

## 1. 数据量化模型

设第 $j$ 个样品的第 $i$ 位评分员在 $n$ 个指标上的得分为 $s_{ijk}$，其总分 $X_{ij}$ 计算如下：

$$
X_{ij} = \sum_{k=1}^{n} s_{ijk}
$$

样品的平均得分 $\bar{X}_j$（期望值）为：

$$
\bar{X}_j = \frac{1}{m} \sum_{i=1}^{m} X_{ij}
$$

*(其中 $m$ 为评分员人数)*

---

## 2. 方差分解模型 (ANOVA)

我们将总变异分解为样品间差异和评分员误差。

**组间平方和 (SSB)**：反映样品真实的质量差异。

$$
SS_{between} = m \sum_{j=1}^{k} (\bar{X}_j - \bar{X}_{grand})^2
$$

**组内平方和 (SSW)**：反映评分员之间的一致性误差。

$$
SS_{within} = \sum_{j=1}^{k} \sum_{i=1}^{m} (X_{ij} - \bar{X}_j)^2
$$

---

## 3. 组内相关系数 (ICC)



通过均方差（Mean Squares）计算信度指标：

1. **组间均方**：$MS_{B} = \frac{SS_{B}}{k-1}$
2. **组内均方**：$MS_{W} = \frac{SS_{W}}{k(m-1)}$

**ICC 定义公式**：

$$
ICC = \frac{MS_{between} - MS_{within}}{MS_{between} + (m-1)MS_{within}}
$$

---

## 4. 判定标准

| ICC 指标范围 | 可靠性等级 |
| :--- | :--- |
| $> 0.75$ | 极高信度 (Excellent) |
| $0.40 - 0.75$ | 中等信度 (Fair to Good) |
| $< 0.40$ | 低信度 (Poor) |
