### 2. 正态性检验（Normality Testing）

在进行推断统计前，需验证样本平均分的分布特性，以决定后续统计方法的适用性。

* **Shapiro-Wilk 检验**：用于检验小样本量下数据是否符合正态分布。其原假设  为：样本数据来自正态分布。
* **D'Agostino's  检验**：通过计算数据的偏度（Skewness）和峰度（Kurtosis）来评估其与正态分布的偏离程度。
* **决策准则**：若计算得到的 -value ，则无法拒绝原假设，认为数据在统计学上符合正态分布特征。
